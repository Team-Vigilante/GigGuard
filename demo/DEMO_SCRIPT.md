## The Script

**0:00 — Intro (Person 1)**
"India has 15 million gig workers. When Swiggy or Zomato
deactivates their account, they have no legal recourse,
no lawyer, no voice. GigGuard changes that — a WhatsApp
chatbot that fights back."

**1:00 — Send WhatsApp message (Person 3)**
- Open WhatsApp on demo phone
- Send "Hi" to sandbox number
- Show the greeting message response on screen
- Say: "Any worker, any phone, no app needed."

**2:00 — Upload screenshot (Person 3)**
- Send the fake Swiggy deactivation screenshot
- Say: "Our OCR agent reads the screenshot instantly"
- Wait for confirmation message to arrive

**3:00 — OCR Confirmation flow (Person 3)**
- Show the formatted confirmation message on phone:
  Platform: Swiggy
  Event: Account Deactivation
  Amount withheld: Rs. 1,840
  Notice provided: No
- Reply YES
- Say: "Worker verifies the data — no hallucinations."

**4:00 — Agent pipeline (Person 2)**
- Show the terminal/logs briefly
- Say: "Three AI agents now work in parallel —
  Parser structures the facts, Researcher finds
  applicable labour laws, Drafter writes the letter."
- Show agent output in terminal
